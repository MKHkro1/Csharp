# 字符串拼接方法 1
```C#
#region 字符串拼接方式1
//之前的算数运算符 知识用来数值类型变量进行数学运算的
//而 string 不存在算数运算符不能计算 但是可以通过+号来进行字符串的拼接
string str = "123";
//其实+号进行字符串拼接
str = str + "456";
Console.WriteLine(str);

str = str + 1; //1会被自动转换成字符串

// 复合运算符 +=
str = "123";
str += "1" + 4 + true;
Console.WriteLine(str);
//都是会被转换成字符串进行拼接

str += 1 + 2 + 3 + 4;// 12310
// 1+2+3+4 会被当成数学运算进行计算 计算完成后再向左边赋值
//1234都是int型，只有10在给str时才发现类型不对进行了类型转换

str += "" + 1 + 2 + 3 + 4;// 1231234
// ""是字符串类型，后面的1+2+3+4都会被当成字符串进行拼接

Console.WriteLine(str);

str = "";
str += 1 + 2 + "" + 3 + 4;
Console.WriteLine(str);
str += 1 + 2 + "" + ( 3 + 4 );
//看字符串和数字的前后顺序，只要碰见字符串则后面的数字都变成字符串，碰不见那就算了
//如果用单引号开始也会被转成数字加入运算而不是字符拼接，而且true既不是1，也不是字符串不能和运算时的＋放一起

int a = '1' + 2 + 3 + 4;// 58
//在C++中，字符常量如'1'实际上是一个整数，代表该字符的ASCII值。ASCII值中，字符'1'的十进制值是49。
Console.WriteLine(a);

str = "123";
str = str + 1 + 2 + 3;
Console.WriteLine(str);//123123
//遵循等号右边从左到右的顺序进行计算，然后把结果赋值给左边（运算符思维）

str = str + ( 1 + 2 + 3 );
//注意：用+号拼接 是用符号的唯一方法 不能用-*/%...
#endregion
```
