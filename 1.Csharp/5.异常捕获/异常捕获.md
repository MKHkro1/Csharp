# 讲解
## 作用
```C#
//引用命名空间
using System;
//命名空间
namespace 模板
{  
    //类
    class Program
    {    //函数
         static void Main(string[] args)
        {   
          #region 作用
		//将玩家输入的内容 存储 string类型的变量（容器）中
		string str = Console.ReadLine();
		//Parse转字符串为 数值类型时 必须合法合规
		int i = int.Parse(str);
		
		//通过异常捕获，可以 避免当代码报错时，造成的程序卡死的情况
		
		#endregion  
        }
    }
    
}

```
## 基本语法
```C#
//引用命名空间
using System;
//命名空间
namespace 模板
{  
    //类
    class Program
    {    //函数
         static void Main(string[] args)
        {   
        	#region 基本语法
	    	//必备部分
			try
			{
			    //希望进行异常捕获的代码块
			    //放进try中
			    //如果try中的代码 报错了 不会让程序卡死
			}
			catch(Exception e) //固定写法，e：当代码出错的时候，这个e会携带上面代码出错的问题进入catch中
			{
			    //如果出错了，会执行 catch中的代码 来捕获异常
			    //（Exception e）具体报错跟踪 通过e得到 具体的错误信息
			}
			//可选部分
			finally
			{
			    //最后执行的代码 不管有没有出错 都会执行其中的代码
			    //目前 可以不用写
			}
			//注意：异常捕获代码基本结构中 不需要加；在里面写代码逻辑时 每一句代码才加；
			#endregion
        }
    }
    
}

```
## 实践
```C#
//引用命名空间
using System;
//命名空间
namespace 模板
{  
    //类
    class Program
    {    //函数
         static void Main(string[] args)
        {   
            #region 实践
			
			try
			{
			    string str2 = Console.ReadLine();
			    int i2 = int.Parse(str2);   
			    Console.WriteLine(i2);
			}
			catch 
			{
			    Console.WriteLine("请输入合法数字");
			}
			//最后可以再加一个循环，报错后让用户重新输入
			//通常异常捕获和if的循环是一起使用的
			//逻辑出错 后续将不会进行执行 不会导致程序的卡死
			
			#endregion
		}
    }
    
}

```
# 实操
1. 请用户输入一个名字，如果输入有误，则提示用户输入错误
2. 提示用户输入姓名、语文、数学、英语成绩。如果输入的成绩有误，则提示用户输入错误，否则将输入的字符串转成整形变量储存
